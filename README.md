# Konversi Suhu (Python)

Program sederhana untuk konversi suhu antar satuan:

- Celsius (C)
- Fahrenheit (F)
- Kelvin (K)

## Cara Menjalankan

1. Clone repo:
   ```bash
   git clone https://github.com/username/konversi-suhu.git
   cd konversi-suhu
   ```

2. Jalankan program:
   ```bash
   python main.py
   ```

3. Contoh output:
   ```
   === KONVERSI SUHU ===
   Masukkan nilai suhu: 100
   Dari satuan (C/F/K): C
   Ke satuan (C/F/K): F
   Hasil: 100C = 212.0F
   ```
